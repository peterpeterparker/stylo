Stylo

Stylo is an open source WYSIWYG project from DeckDeckGo, an editor for presentations, documents and blog posts. With its particular design and customizable plugin architecture, it fits well an interactive user experience.
The project is under development and published in ​alpha​ version, it can already be used by anyone but still needs improvements. Your contribution on GitHub is most welcome!

Interactive design üéØ

A clean design and an intuitive user experience are the leitmotiv of Stylo. Serving users the feature they need when they edit their content in the most straightforward way possible is the goal of the project.

Customizable üí™

Based on a set of plugins that can be fully customized, enabling additional functionality or disabling default features is made easy.

Framework agnostic üòé

The editor is a lightweight (in comparison to other editors - 32kb gzipped) and framework agnostic library of reusable components.

Its code is agnostic too - i.e. no 3rd party runtime dependencies - and works across all modern browsers.

The code is published under MIT license and contribution are most welcome!

Future Proof üöÄ

Even though at the moment (alpha version) few actions still relies on execCommand to apply styles (e.g. bold, italic) - the core of Stylo does not. It controls what happens on every mutation and avoids the need to use this deprecated API and is therefore designed for the future.
